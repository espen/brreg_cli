# Installasjon

```shell
$ git clone https://github.com/espen/brreg.git
$ cd brreg
$ brreg.rb -h
```

# I bruk

```shell
brreg [options]
-n, --orgnr ORGNR                Organisasjonsnummer
-q, --query QUERY                Firmanavn
-d, --domain DOMAIN              Domenenavn (kun .no)
-v, --version                    Versjon
```
